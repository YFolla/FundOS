# Companies Index

This is the primary working area for the repo.

## Retrieval Path

1. Open [companies-index.yaml](/Users/yannickfolla/CascadeProjects/TeamOS/companies/companies-index.yaml).
2. Find the company slug.
3. Open only the files linked from that entry.

## Canonical Company Files

Each company folder should usually contain:

- `company.md`
- `people.md`
- `decision-log.md`
- `interactions/`
- `diligence/`
- `memos/`

Portfolio companies can also include `portfolio-plan.md`.
