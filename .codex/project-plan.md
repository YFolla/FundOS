# Project Plan

## Goal

Create a White Star Capital TeamOS starter repository that is:

- optimized for Codex retrieval
- portable to Claude
- seeded with realistic example objects
- accompanied by a practical ingestion plan for existing firm systems

## Current Scope

1. Create a root operating map and thin agent adapters.
2. Create top-level folders and indexes for team, firm, companies, markets, funds, and operations.
3. Seed the repo with fictional examples that illustrate sourced, diligence, and portfolio workflows.
4. Add a seeding playbook that explains how to migrate from Box and adjacent systems.
5. Add an automated verification script.

## Out Of Scope

- real data ingestion
- automated Box export tooling
- CRM synchronization
- access-control model across multiple repos

## Next Logical Iterations

1. Replace fictional objects with live firm examples.
2. Add portfolio support and IC memo templates that match real internal workflow.
3. Add scripts to transform exported metadata into the YAML indexes.
4. Split restricted materials into a companion private repo if needed.
