# Firm Overview

This file should become the concise, high-signal summary of the firm.

## Template Positioning

For this starter repo, assume White Star Capital is operating as a global, multi-stage technology investment firm and that this repo needs to serve:

- sourcing
- diligence
- IC decisions
- portfolio support
- LP and fund reporting

## Recommended Contents For The Real Version

- firm mission and positioning
- current fund family and mandates
- stage focus by geography
- key vertical theses
- decision-making forums and owners
- definitions for sourced, active diligence, pass, watchlist, and portfolio

## Authoring Rule

Keep this short. Detailed theses belong under `firm/investment-theses/` or `markets/`.
